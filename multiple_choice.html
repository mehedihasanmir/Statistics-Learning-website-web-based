Multiple choice for basic Statistics

  1. What is the median of the following data set: 12, 15, 18, 20, 22?

    1. 18
    2. 19
    3. 20
    4. 22

  2. The range of a data set is:

    1. The average value
    2. The difference between the smallest and largest values
    3. The most frequently occurring value
    4. The sum of all values

  3. What does the standard deviation measure?

    1. The spread of data
    2. The center of data
    3. The average of data
    4. The mode of data

  4. If the mean of a data set is 25 and the sum of the values is 200, how many values are in the data

    1. 5
    2. 8
    3. 10
    4. 12

  5. The mode of a data set is:

    1. The average value
    2. The middle value
    3. The most frequently occurring value
    4. The sum of all values

  6. The interquartile range (IQR) is defined as:

    1. The difference between the mean and median
    2. The range divided by the mean
    3. The range between the first and third quartiles
    4. The standard deviation divided by the mean

  7. In a normal distribution, what percentage of data falls within two standard deviations of the mean

    1. 34%
    2. 50%
    3. 68%
    4. 95%

  8. The correlation coefficient measures:

    1. The strength and direction of a linear relationship between two variables
    2. The spread of data
    3. The mode of a data set
    4. The probability of an event

  9. The formula for calculating variance is:

    1. Mean / Range
    2. Standard Deviation / Mean
    3. Sum of Squares / Number of Values
    4. Range / Mean

  10. If the p-value in a hypothesis test is greater than the significance level (Î±), what should you do?

    1. Reject the null hypothesis
    2. Fail to reject the null hypothesis
    3. Accept the null hypothesis
    4. Double-check the data

  11. What is the purpose of a confidence interval?

    1. To estimate the population parameter
    2. To test hypotheses
    3. To determine the standard deviation
    4. To calculate the z-score

  12. The chi-square test is used to:

    1. Compare means of two independent groups
    2. Test for a significant relationship between two categorical variables
    3. Determine the correlation coefficient
    4. Test the difference between paired observations

  13. What does a z-score of -2 signify in a standard normal distribution?

    1. The data point is below the mean by two standard deviations
    2. The data point is above the mean by two standard deviations
    3. There is no significant information in the z-score
    4. The data point is two times the mean

  14. If the mean of a data set is 50 and the standard deviation is 10, what is the coefficient of variation?

    1. 0.1
    2. 0.2
    3. 0.5
    4. 1

  15. The probability of rolling a total of 7 on two six-sided dice is:

    1. 1/6
    2. 1/8
    3. 1/12
    4. 1/36

  16. In a box-and-whisker plot, the box represents:

    1. The first and third quartiles
    2. The median
    3. The range
    4. The mean

  17. Which of the following is NOT a measure of central tendency?

    1. Mean
    2. Median
    3. Mode
    4. Range

  18. What is the formula for calculating the mean (average) of a data set?

    1. Sum of values / Number of values
    2. Sum of squares / Number of values
    3. Range / Number of values
    4. Standard Deviation / Number of values

  19. A confidence level of 95% in a confidence interval means:

    1. There is a 95% probability that the population parameter is within the interval
    2. The sample mean is within 95% of the population mean
    3. 95% of the data falls within the interval
    4. The interval contains 95% of the sample data

  20. If the correlation coefficient is 0, it indicates:

    1. A strong positive correlation
    2. A strong negative correlation
    3. No correlation
    4. Perfect correlation
